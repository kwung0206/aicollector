# AIVideo

생성형 AI 영상 결과물을 더 빠르게 찾고, 공유하고, 재활용할 수 있도록 만든 AI 영상 컬렉션 플랫폼입니다.

## 프로젝트 소개

생성형 AI로 영상을 만드는 과정은 시간이 오래 걸리고, 어렵게 만든 결과물이 원하는 형태가 아닐 때도 많습니다.  
AIVideo는 이런 문제를 줄이기 위해 만들어졌습니다.

여러 사용자가 생성한 AI 영상을 업로드하고 공유할 수 있게 하여,
다른 사용자가 비슷한 결과물을 다시 처음부터 만들기보다
이미 존재하는 결과물을 탐색하고 다운로드해 더 빠르게 원하는 영상에 도달할 수 있도록 돕습니다.

## 핵심 기능

### 1. 카테고리 기반 AI 영상 정리
- 업로드 시 태그를 선택하여 영상을 체계적으로 분류
- 영상 모음 페이지에서 필터링으로 원하는 영상을 빠르게 탐색
- 좋아요, 싫어요, 조회수 기준으로 대중 반응 확인 가능
- 업로드된 영상을 이미지 프레임 단위로 분석해 특징 데이터를 저장

### 2. 프롬프트 기반 유사 영상 추천
- 사용자가 입력한 프롬프트를 기반으로 예상 영상 특징 분석
- 기존 영상에서 추출한 특징과 비교하여 유사한 결과물 추천
- 추천 정확도를 사용자가 직접 조절 가능

### 3. 유해 영상 필터링
- Google Cloud Video Intelligence API를 활용해 유해 영상 여부 판별
- 선행 학습된 머신러닝 모델을 통해 객체, 장소, 동작 등을 분석
- 판별된 영상은 관리자가 검토 후 차단 해제 또는 삭제 가능

## 동작 방식

1. 사용자가 AI 영상을 업로드합니다.
2. 서버가 영상을 프레임 이미지로 추출합니다.
3. 추출된 이미지와 영상 정보를 바탕으로 특징 데이터를 생성합니다.
4. 사용자는 태그, 반응 지표, 프롬프트 검색을 통해 원하는 영상을 탐색합니다.
5. 유해성 판별을 통해 플랫폼의 안전성을 관리합니다.

## 사용 기술

- **Frontend**: React, Vite
- **Backend**: Spring Boot, Gradle
- **AI 분석**: ChatGPT API
- **영상 처리**: FFmpeg
- **유해성 판별**: Google Cloud Video Intelligence API

## 기대 효과

- AI 영상 생성 결과물을 재사용 가능한 자산으로 축적
- 원하는 스타일의 영상을 더 빠르게 찾을 수 있는 검색 경험 제공
- 반복 생성 비용과 시간을 절감
- 커뮤니티 기반의 AI 영상 탐색 및 공유 환경 구축
