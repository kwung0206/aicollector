02

핵심 기능

AI 면접

이력서와 GitHub 정보를 기반으로 맞춤형 질문을 생성하고 면접 종료 후 평가를 제공합니다.

채용 공고

자소서 샘플

직무별 샘플을 참고하고 사용자가 작성한 자기소개서를 AI가 평가합니다.

05

시스템 구성

서버 구성

CentOS 9 환경에서 GitHub 코드를 가져오고 Docker로 프론트엔드, 백엔드, 데이터베이스, 이메일 서버 컨테이너를 구성했습니다.

보안/운영

Nginx, Let’s Encrypt + Certbot, iptables, Docker 내부망, DKIM 기반 발신 전용 이메일 서버를 활용했습니다.

06

데이터 흐름

도메인 접속: 가비아 도메인 ↔ 공인 IP
포트 포워딩/NAT: 80·443 요청을 서버 노트북으로 전달
Front 컨테이너: Docker + Nginx
Backend 컨테이너: Spring Boot API 처리
Email/DB: DKIM 발신 전용 이메일 서버 + Oracle
보안: iptables 차단, HTTPS 인증, Docker 내부망 구성
